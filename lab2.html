Лабораторная работа 2

Описание проекта

В рамках второй лабораторной было создано веб приложение, собирающее акции на путешествия с источников. Собранные акции заносятся в базу данных. Далее данные отображаются на сайте. Пользователь может предложить свои акции, воспользовавшись формой. Предложеннеы Акции попадают в базу данных с соответсвующей отметкой. Пока они не будут обработаны администратором, они не будут отображаться.

Для управления базой данных и обновлением парсеров была создана панель администратора.

используемые технологии

html, css, js, jquery, bootstrap, php, mysql
